# Food Store - Sistema de Gestion de Pedidos de Comida

Trabajo Practico Integrador de Programacion II (UTN - Tecnicatura Universitaria
en Programacion a Distancia).

Aplicacion de consola en Java que permite gestionar **categorias, productos,
usuarios y pedidos** mediante operaciones CRUD desde un menu de texto. Toda la
informacion se almacena **en memoria** durante la ejecucion usando Colecciones
(no hay base de datos ni login).

## Requisitos

- Java 21 (JDK 21 o superior).
- Opcional: NetBeans / IntelliJ / VS Code con soporte Java (el proyecto incluye
  configuracion de NetBeans).

## Como compilar y ejecutar

### Desde un IDE (NetBeans)

1. Abrir el proyecto.
2. Ejecutar la clase `integradoR.prog2.Main`.


Al iniciar se cargan datos de ejemplo (categorias, productos, usuarios y un
pedido) para poder probar el sistema de inmediato.


Separacion por capas: **Entities** (modelo), **Service** (validaciones y reglas
de negocio, mantienen las colecciones y los contadores de id) y **View/Main**
(interaccion con el usuario).
